# Konnekt Official

This GitHub organization hosts all repositories related to **Konnekt**, a modern social media and messaging platform developed as part of the **ST4005CMD - Integrative Project** at **Softwarica College of IT and E-Commerce**.

---

## About Konnekt

Konnekt is a **Java-based social media and messaging application** designed to provide:

- Real-time messaging (1-on-1 and group chats)  
- Audio and video calls  
- Social feed with posts, comments, and likes  
- User profiles, settings, and notifications  
- Community and membership features  
- Shorts (short videos) and content sharing  
- Admin panel for platform management  
- Followers and following system  

The project is developed using:

- **Java** (main language, all backend and frontend logic)  
- **NetBeans** (IDE for development)  
- **MySQL** (database for storing users, messages, media, etc.)  
- **WebSocket / WebRTC frameworks** (for real-time messaging and audio/video calls)  
- **Git & GitHub** (version control)  
- **Figma** (design and prototyping)  
- **AWS** (deployment and hosting)  
- **Trello** (project management)

> **Note:** The project uses only **Java** in the repositories as per the course requirement.
